# LichSOMA's Speaker Selector

> **이 모듈은 생성형 AI의 도움으로 제작되었습니다.**

## 호환성

- **Foundry VTT**: v14 이상 (`module.json` 기준)

## 소개

Foundry VTT의 채팅 시스템을 확장하는 모듈입니다. 토큰 선택과 독립적인 스피커 선택, 채팅 커스터마이징, 나레이터 모드, 채팅 로그 HTML 내보내기·아카이브 등을 제공합니다.

## 주요 기능

### 스피커 선택

- 토큰 선택과 독립적으로 말하는 캐릭터를 선택할 수 있습니다.
- 채팅 입력 영역 근처의 드롭다운에서 스피커 선택.
- OOC(Out of Character) 모드 지원(설정 조합에 따라 옵션 표시).

### 스피커 설정 (GM)

- 채팅 입력 창의 설정 버튼으로 사용 가능한 액터 관리.
- 드래그 앤 드롭으로 액터 추가·제거, 검색.

### 감정 시스템

- 액터별 감정 표정 설정 및 채팅 포트레잇 반영.

### 채팅 커스터마이징

- **포트레잇**: 크기·표시 옵션.
- **폰트**: 헤더·메시지·나레이터 폰트, 크기, 굵기.
- **한자 전용 폰트**: 헤더·메시지에 한자 전용 패밀리 지정 가능.
- **유저 색상**: 발신자 이름에 유저 색 적용.
- **채팅 머지**: 같은 스피커 연속 메시지에서 헤더 생략(설정으로 켜기).
- **마크다운·루비**: `~취소선~`, `*이탤릭*`, `**볼드**`, `[[본문|루비]]` 등.

### 사이드바·채팅 입력 크기 조절

- 사이드바 **왼쪽 가장자리**를 드래그해 너비 조절(저장된 최소 너비 이상).
- 채팅 입력 영역 **상단 가장자리**를 드래그해 입력 칸 높이 조절(`--chat-input-height`).

### 채팅 입력 — 이미지 삽입

- 채팅 컨트롤(저장·비우기 버튼 근처)에 **이미지 삽입** 버튼을 제공하고, 기본 ProseMirror 상단 포맷 메뉴는 숨겨 두어 UI를 단순화합니다.

### 나레이터 모드 (GM)

- 나레이터 버튼·타이핑 효과·사운드·스타일 설정.
- `/desc …` 로 나레이터 카드 형태 출력 등.

### 채팅 로그 HTML 내보내기

- 채팅 패널의 저장 아이콘 동작을 가로채 **단일 HTML 파일**로 저장.
- 모듈 기본 스타일(`lichsoma-chat-log-export.css`) 인라인 포함.
- **설정**: 상대 경로 기준 URL(`chatLogExportBasePath`), 이미지 **Base64 포함 여부**, 내보낸 HTML에서 **주사위 툴팁 표시 여부**, **커스텀 CSS**(월드 저장 + 설정 화면에서 코드 편집 메뉴).
- **참고**: 메시지 수가 매우 많으면 브라우저에서 처리 시간·메모리 부담이 커질 수 있습니다. Base64 옵션은 이미지가 많을수록 출력이 무거워집니다.

### DX3rd (`dx3rd-fvtt`) 월드 — 내보내기 스타일

- 게임 시스템이 **dx3rd-fvtt**일 때, HTML 내보내기에 `styles/apps/lichsoma-chat-dx3rd.css` 내용을 **내보낸 HTML에 맞게 스코프 보정(descope)** 하여 추가로 합칩니다. (인게임 채팅용 DX 스타일과 내보내기 결과를 맞추기 위함.)

### 채팅 로그 아카이브

- 과거 로그 폴더 탐색·검색.

### 타이핑 표시

- 다른 사용자 입력 중 표시.

## 설치

GitHub 릴리스의 `module.json` manifest URL로 설치하거나, 모듈 폴더를 `Data/modules` 아래에 두고 활성화합니다.

## 설정

**게임 설정 → 모듈 설정**에서 조정합니다.

## 사용 팁

- 스피커·감정·나레이터는 세션 연출에 맞게 조합해 사용하세요.
- 긴 캠페인 로그는 **주기적으로 HTML로 나누어 저장**하면 브라우저 부담을 줄일 수 있습니다.
